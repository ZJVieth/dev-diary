### Starting a project with vite
```bash
npm init vite@latest
cd <project-name>
npm install
```
or
```bash
npm create @vitejs/app <project-name>
cd <project-name>
npm install
```

### Running the Dev Server
```bash
npm run dev
http://localhost:3000
```
